フォントを変更してみる#

Google Fonts をつかってみる

  1. PaperMod の拡張用の機能を使用する
    • ヘッダ拡張として layouts/partials/extend_head.html を作成する
    • CSS 拡張として assets/css/extended/custom.css を作成する
  2. Google Fonts から好きなものを選択して、右上のボタンを押す
    • Download famiy ではない
  3. Use on the web の文字列をコピーして、作成したファイルにそれぞれの文字列を張り付ける
    <link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Yomogi&display=swap" rel="stylesheet">

    body {
  font-family: 'Yomogi', cursive;
}
